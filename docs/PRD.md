# 咖啡厅经营小游戏 — 产品需求文档（PRD）

> 版本：v1.0 | 日期：2026-07-02 | 状态：初稿

---

## 1. 产品概述

### 1.1 产品定位

一款**单人休闲咖啡厅经营模拟游戏**，面向喜欢模拟经营、放松解压的玩家。玩家扮演咖啡厅老板，从一间小店起步，通过制作咖啡、服务顾客、装修店铺、招聘员工，逐步打造属于自己的梦想咖啡厅。

### 1.2 目标用户

| 维度 | 描述 |
|------|------|
| 年龄 | 12 - 35 岁 |
| 偏好 | 模拟经营、养成、休闲、治愈系 |
| 使用场景 | 碎片时间、通勤、睡前放松 |
| 核心诉求 | 不肝不氪、操作简单、有成长感、自由度高 |

### 1.3 核心体验目标

- 🎨 **视觉治愈**：卡通可爱风格，温暖色调，细腻的场景与角色动画
- ☕ **轻松解压**：无强制任务、无时间压迫，玩家按照自己的节奏游玩
- 🏠 **归属感**：从空荡荡的小店面到热闹温馨的咖啡厅，见证成长
- 🎯 **持续动力**：丰富的收集、装修、成就系统，让玩家有长期目标

## 2. 游戏世界观与美术风格

### 2.1 世界观

故事发生在一座悠闲的海滨小镇「暖阳镇」。镇上有一条安静的商业街，街角有一间被遗忘的小店。玩家继承（或租下）了这间小店，开启咖啡厅经营之旅。

### 2.2 美术风格

| 维度 | 描述 |
|------|------|
| 整体风格 | 卡通Q版、手绘质感、温暖治愈 |
| 色调 | 暖棕色系为主（咖啡、木质），搭配奶油白、薄荷绿、樱花粉 |
| 角色设计 | 二头身Q版小人，圆润可爱，表情丰富 |
| 场景 | 2.5D 等距视角（类似《动物森友会》/《欢乐咖啡厅》） |
| UI | 圆角卡片、手写字体、柔和阴影、微动效 |
| 参考游戏 | 《动物森友会》《猫咪咖啡厅》《Cats & Soup》《海绵宝宝：大闹蟹堡王》 |

### 2.3 音乐与音效

- **背景音乐**：轻快舒缓的 Lo-fi / Jazz Hop，随时间与天气变化（早晨轻快、夜晚柔和、雨天慵懒）
- **音效**：
  - 咖啡机萃取声、奶泡打发声、杯碟碰撞声
  - 收银机"叮"声、顾客满意的轻叹声
  - UI 点击的柔和反馈音

## 3. 核心玩法

### 3.1 玩法总览

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   制作咖啡/食物  ──→  服务顾客  ──→  赚取金币/钻石    │
│         ↑                              ↓            │
│         │                        ┌──────────┐       │
│         │                        │ 扩建店铺  │       │
│         │                        │ 购买设备  │       │
│         └──── 解锁新食谱 ←───────│ 装修布置  │       │
│                                  │ 雇佣员工  │       │
│                                  │ 采购原料  │       │
│                                  └──────────┘       │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 3.2 玩法一：咖啡/食物制作（核心循环）

玩家通过简单的交互制作各种咖啡和甜品。

#### 制作流程

1. **接单**：顾客进店，头顶气泡显示想要的商品
2. **制作**：点击咖啡机/设备，进入制作界面
   - **简单模式**：点击 / 滑动完成步骤（如：磨豆→萃取→打奶泡→拉花）
   - 每步有 2-3 秒等待时间（可通过升级设备缩短）
3. **出餐**：将成品拖拽给顾客
4. **结算**：获得金币 + 经验值，顾客可能给好评/小费

#### 食谱体系

| 类别 | 示例 | 解锁方式 |
|------|------|---------|
| 经典咖啡 | 美式、拿铁、卡布奇诺、摩卡 | 初始解锁 / 等级解锁 |
| 特调饮品 | 焦糖玛奇朵、抹茶拿铁、生椰拿铁 | 等级 + 金币解锁 |
| 冷饮系列 | 冰美式、冰拿铁、冷萃、气泡咖啡 | 购买设备后解锁 |
| 甜品 | 提拉米苏、芝士蛋糕、马卡龙、华夫饼 | 购买烤箱后解锁 |
| 季节限定 | 樱花拿铁（春）、西瓜冷萃（夏）、南瓜拿铁（秋）、姜饼拿铁（冬） | 季节活动解锁 |
| 隐藏食谱 | 秘密配方（特定组合触发） | 探索 / 顾客线索 |

#### 品质系统

- 每杯咖啡有 **1-5 星** 品质评分
- 影响因素：设备等级、原料品质、操作准确度、员工技能
- 高品质咖啡获得更多金币 / 小费 / 好评

### 3.3 玩法二：顾客系统

#### 顾客类型

| 类型 | 特征 | 偏好 | 耐心值 |
|------|------|------|--------|
| 👔 上班族 | 早上出现，赶时间 | 美式/拿铁，快速出餐 | ⭐ |
| 👩‍🎓 学生 | 下午出现，预算有限 | 甜口咖啡+甜品套餐 | ⭐⭐⭐⭐ |
| 👵 退休老人 | 全天随机，悠闲 | 经典款，喜欢聊天 | ⭐⭐⭐⭐⭐ |
| 📸 网红博主 | 看重颜值 | 特调饮品，要求拉花/摆盘 | ⭐⭐⭐ |
| 💼 商务人士 | 高端消费 | 单品咖啡，对品质敏感 | ⭐⭐ |
| 🎨 文艺青年 | 喜欢安静角落 | 手冲/冷萃，对氛围有要求 | ⭐⭐⭐⭐ |
| 🐱 特殊顾客 | 随机出现 | 特殊订单，奖励丰厚 | ⭐⭐⭐⭐⭐ |

#### 顾客行为

- 顾客进店→排队→看菜单→点单→等待→用餐→结账→离开
- 等待过久会生气离开（影响店铺口碑评价，不扣金币）
- 满意度高会留下小费、成为回头客、推荐新顾客
- 顾客有自己的作息时间（早餐高峰 7-9 点、下午茶 14-16 点、晚间 19-21 点）

### 3.4 玩法三：店铺经营与升级

#### 金币与钻石

| 货币 | 获取方式 | 用途 |
|------|---------|------|
| 🪙 金币 | 出售商品、完成任务、成就奖励 | 购买设备、装修、原料、发工资 |
| 💎 钻石 | 完成特殊订单、成就、每日签到 | 加速建造、稀有装饰、特殊道具 |

> 注：本游戏不设计内购，所有货币均通过游戏内行为获得。

#### 店铺等级

| 等级 | 名称 | 解锁内容 |
|------|------|---------|
| Lv1 | 街角小店 | 基础咖啡机、2张桌子、10个食谱 |
| Lv5 | 社区热门 | 甜品柜台、4张桌子、20个食谱、1名员工 |
| Lv10 | 网红打卡店 | 手冲区、露台座位、35个食谱、3名员工 |
| Lv20 | 暖阳镇名片 | 全部区域、定制菜单、5名员工、无限创意 |

#### 设备系统

| 设备 | 作用 | 可升级属性 |
|------|------|-----------|
| 意式咖啡机 | 制作浓缩咖啡 | 速度、品质加成 |
| 磨豆机 | 研磨咖啡豆 | 速度、品质加成 |
| 奶泡机 | 制作奶泡（拿铁/卡布必备） | 品质加成 |
| 手冲壶套装 | 制作手冲/冷萃 | 品质加成、解锁新食谱 |
| 烤箱 | 制作甜品 | 速度、解锁新食谱 |
| 冷藏柜 | 存放牛奶/奶油/蛋糕 | 容量 |
| 收银机 | 结账 | 结算速度 |

### 3.5 玩法四：自由装修

玩家可以自由布置咖啡厅内部，这是游戏的重要乐趣来源。

#### 装修系统

- **网格系统**：店内划分为网格，家具放置在网格上
- **自由拖拽**：拖拽家具到任意位置，旋转方向
- **预览模式**：布置前可预览效果
- **保存方案**：支持多套装修方案保存与切换

#### 可装饰元素

| 类别 | 内容 |
|------|------|
| 🪑 桌椅 | 双人桌、四人桌、吧台座、沙发区、露台座 |
| 💡 灯具 | 吊灯、壁灯、落地灯、台灯、氛围串灯 |
| 🪴 绿植 | 盆栽、挂壁植物、花瓶、小花园 |
| 🖼️ 墙面 | 墙纸、挂画、照片墙、黑板菜单、钟表 |
| 🪟 窗户 | 窗框样式、窗帘、窗台装饰 |
| 🚪 门 | 店门样式、招牌 |
| 🎨 地板 | 木地板、瓷砖、地毯 |
| 🎵 音响 | 点唱机/音响设备 |
| 🐱 宠物 | 店猫，在店内自由走动，吸引顾客 |

#### 主题风格

提供成套的主题装修风格，一键切换：
- 🪵 复古木质风
- 🌿 森系自然风
- 🎀 甜美少女风
- 🏭 工业极简风
- 🎄 节日主题（圣诞/万圣/春节）
- 🌊 海滨度假风

### 3.6 玩法五：员工系统

#### 可雇佣员工

| 职位 | 作用 | 可升级属性 |
|------|------|-----------|
| ☕ 咖啡师 | 自动制作咖啡 | 速度、品质、可同时制作数量 |
| 🍰 甜品师 | 自动制作甜品 | 速度、品质 |
| 🏃 服务员 | 送餐、收拾桌子 | 速度、一次可端盘数 |
| 🧹 清洁工 | 清洁桌子、扫地 | 速度、影响店铺整洁度 |
| 🎵 驻唱歌手 | 提升顾客心情，延长等待耐心 | 演奏水平 |

#### 员工养成

- 每个员工有 **等级**（影响效率）和 **心情值**（影响工作质量）
- 心情值通过发工资、发奖金、安排休息维持
- 员工升级需要经验值，获得方式为持续工作
- 高级员工解锁特殊技能（如咖啡师解锁拉花技能，提升小费概率）

### 3.7 玩法六：供应与库存

#### 原料采购

| 原料 | 供应商 | 品质等级 |
|------|--------|---------|
| 咖啡豆（埃塞/哥伦比亚/巴西/云南） | 豆叔 | 普通/精品/庄园级 |
| 牛奶/燕麦奶/椰奶 | 农场直供 | 普通/有机 |
| 糖浆（焦糖/榛果/香草/玫瑰） | 甜蜜屋 | 普通/进口 |
| 面粉/黄油/奶油（甜品用） | 烘焙坊 | 普通/进口 |
| 杯子（纸杯/瓷杯/外带杯） | 杂货铺 | 普通/精美 |

#### 库存管理

- 每种原料有库存上限（可通过升级仓库扩大）
- 库存不足时无法制作对应商品
- 原料会过期（生鲜类24小时游戏时间 / 干货不限）
- 可设置自动补货阈值

### 3.8 玩法七：特殊事件与任务

#### 随机事件

| 事件 | 触发条件 | 效果 |
|------|---------|------|
| 📰 美食博主探店 | 店铺等级≥5，随机 | 大量客流，评价影响后续客流 |
| 🎂 顾客生日 | 回头客来访N次 | 制作生日特饮，获得钻石 |
| 🌧️ 雨天 | 随机天气 | 户外座位不可用，但热饮销量翻倍 |
| 🐱 流浪猫来访 | 完全随机 | 可收养为店猫 |
| 🏆 咖啡大赛 | 每月一次 | 参赛赢取稀有奖励 |

#### 日常任务

- 每日目标：卖出 X 杯咖啡 / 接待 X 位顾客 / 获得 X 好评
- 完成获得金币、经验、钻石

#### 成就系统

| 成就分类 | 示例 |
|---------|------|
| 经营成就 | 累计收入10万金币、连续营业100天 |
| 制作成就 | 制作1000杯咖啡、所有食谱满星 |
| 顾客成就 | 100位回头客、获得100次小费 |
| 装修成就 | 收集100件家具、解锁全部主题 |
| 员工成就 | 培养3名满级员工 |
| 隐藏成就 | 特定组合触发（如连续做出10杯完美咖啡） |

### 3.9 玩法八：社交与分享（轻量）

- **拜访好友**：查看好友的咖啡厅布局（仅展示，无直接互动）
- **咖啡厅名片**：生成一张精美名片图，展示店铺评分/装修/特色
- **赠送礼物**：每日可给好友赠送少量原料
- **排行榜**：小镇咖啡厅排名（纯展示，无竞争压力）

## 6. 技术方案建议

### 6.1 技术选型

| 层级 | 技术选择 | 理由 |
|------|---------|------|
| 游戏引擎 | **PixiJS** 或 **Phaser 3** | 2D Web 游戏成熟方案，性能好，生态丰富 |
| 前端框架 | **React + TypeScript** | UI 层用 React，游戏层用 Canvas，组合灵活 |
| 渲染方式 | Canvas（游戏场景）+ DOM（UI面板） | 兼顾性能与UI开发效率 |
| 状态管理 | **Zustand** | 轻量、TypeScript友好、适合游戏状态 |
| 动画 | **GSAP** + CSS Transition | 细腻的过渡动画 |
| 数据持久化 | **IndexedDB** (Dexie.js) | 本地存档，离线可玩 |
| 构建工具 | **Vite** | 快速开发体验 |
| 测试 | Vitest + Playwright | 单元测试 + E2E |

### 6.2 为什么选 Web 技术栈

- ✅ 跨平台（PC / 平板 / 手机浏览器）
- ✅ 无需安装，即开即玩
- ✅ 开发迭代快，热更新
- ✅ 可打包为 PWA，离线游玩
- ✅ 方便后续扩展为 Electron 桌面应用

### 6.3 资源规格

| 资源类型 | 规格 |
|---------|------|
| 场景素材 | 等距 PNG 精灵图，建议 2048×2048 图集 |
| 角色动画 | Spine / 帧动画精灵图 |
| UI图标 | SVG 矢量图标 |
| 音频 | MP3/OGG，H5 Audio API |

### B. 数值设计原则

- **前期（Lv1-5）**：快速成长，每 5-10 分钟有成就感
- **中期（Lv6-15）**：稳定节奏，每 20-30 分钟有大进展
- **后期（Lv16+）**：深度策略，装修/员工/食谱的组合优化
- **无金币惩罚设计**：服务失败不扣金币，只损失潜在收益（机会成本）；极度不满可能影响店铺口碑评价，但不影响货币余额。鼓励尝试而非害怕失败
